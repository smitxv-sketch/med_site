# Технические требования и Интеграции

## 1. Интеграция с МИС (Медицинская Информационная Система)
- **Синхронизация цен:** Цены на услуги обновляются по API из МИС.
- **Связь:** Услуги на сайте привязываются к услугам в МИС через уникальный артикул.
- **Врачи:** У каждого врача на сайте указан артикул из МИС.
- **Расписание:** Сервис онлайн-записи (React) забирает расписание из МИС по артикулу врача.

## 2. Миграция данных (WordPress & MODX -> Strapi)
- **Челябинск (WordPress):** Возможность выгрузки любых данных. Потребуются скрипты миграции.
- **Санкт-Петербург (MODX):** Потребуются скрипты миграции.
- **Стратегия:** На этапе разработки использовать скрипты синхронизации для наполнения нового сайта реальными данными.

## 3. SEO и Внешние площадки
- **Яндекс.Врачи:** Генерация XML-фидов по каждому городу.
- **Структура фида:** Включает все необходимые поля со страниц врачей (имя, специальность, стаж, фото, цена приема).

## 4. Аналитика (Яндекс.Метрика)
*Ожидается выгрузка данных для анализа текущего поведения пользователей.*

**Что нужно запросить:**
- Популярные страницы входа (Landing Pages).
- Показатель отказов (Bounce Rate) на мобильных устройствах.
- Пути конверсии (Как пользователи доходят до записи).
- Самые посещаемые страницы врачей и услуг.
