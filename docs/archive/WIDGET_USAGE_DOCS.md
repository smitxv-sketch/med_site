# Руководство по эксплуатации виджета онлайн-записи (Widget Usage Docs)

Этот документ описывает способы интеграции и использования React-виджета онлайн-записи на сайтах клиник (в частности, на базе WordPress) и других платформах.

Виджет спроектирован как независимое Isomorphic/Micro-frontend приложение. Он может работать как в полноэкранном режиме (по прямой ссылке), так и встраиваться в любые страницы через `<iframe>` или веб-компоненты.

## 1. Режимы работы виджета (Routing & Views)

Виджет поддерживает маршрутизацию через URL-параметры (Query Parameters). Это позволяет гибко управлять тем, что видит пользователь при открытии страницы.

### 1.1. Полный каталог (Default View)
Отображает полный список специальностей, врачей и услуг с возможностью поиска и фильтрации.
- **URL:** `https://widget.clinic.ru/` (без параметров)
- **Сценарий:** Главная кнопка "Записаться онлайн" в шапке сайта.

### 1.2. Карточка конкретного врача (Doctor View)
Сразу открывает профиль конкретного врача с его расписанием, минуя шаг выбора специальности.
- **URL:** `https://widget.clinic.ru/?doctorId=12345` (где `12345` — это ID врача из МИС)
- **Сценарий:** Кнопка "Записаться к этому врачу" на персональной странице врача в WordPress.
- **Отображение:** Виджет скрывает лишний UI (поиск, фильтры) и показывает только ФИО, фото, регалии и сетку слотов этого врача.

### 1.3. Список врачей по специальности (Specialty View)
Открывает список всех врачей выбранной специальности (с их ближайшими слотами).
- **URL:** `https://widget.clinic.ru/?specialtyId=789` (где `789` — ID специальности из МИС/CMS)
- **Сценарий:** Кнопка "Записаться" на странице услуги (например, "Консультация кардиолога").
- **Отображение:** Виджет показывает только кардиологов, скрывая другие специальности.

### 1.4. Запись на конкретную услугу (Service View)
Открывает список врачей, оказывающих конкретную услугу.
- **URL:** `https://widget.clinic.ru/?serviceId=456` (где `456` — ID услуги из МИС)
- **Сценарий:** Кнопка "Записаться" рядом с ценой на конкретную процедуру (например, "УЗИ брюшной полости").

## 2. Способы интеграции на сайт (Integration Methods)

### Способ А: Прямая ссылка (Standalone)
Самый простой и надежный способ. Кнопка на сайте просто уводит пользователя на отдельный домен/поддомен виджета.
```html
<!-- На странице врача (ID 12345) -->
<a href="https://widget.clinic.ru/?doctorId=12345" target="_blank" class="btn-book">
  Записаться на прием
</a>
```
**Плюсы:** Нет конфликтов стилей (CSS) с основным сайтом, работает на мобильных устройствах идеально.

### Способ Б: Встраивание через Iframe (Embedded)
Виджет открывается во всплывающем окне (Modal/Popup) поверх текущей страницы сайта или встраивается прямо в контент.
```html
<!-- Вставка на страницу специальности -->
<div class="widget-container">
  <iframe 
    src="https://widget.clinic.ru/?specialtyId=789&theme=light" 
    width="100%" 
    height="600px" 
    frameborder="0"
    allow="geolocation"
  ></iframe>
</div>
```
**Плюсы:** Пользователь не покидает сайт клиники.
**Минусы:** Требует настройки `postMessage` для изменения высоты iframe при переключении шагов внутри виджета (чтобы не было двойных скроллов).

### Способ В: Веб-компонент (Advanced)
Виджет компилируется как единый JS-бандл и подключается на сайт как кастомный HTML-тег.
```html
<script src="https://widget.clinic.ru/bundle.js"></script>

<!-- На странице врача -->
<clinic-booking-widget doctor-id="12345" theme="dark"></clinic-booking-widget>
```
**Плюсы:** Глубокая интеграция, виджет выглядит как часть сайта.
**Минусы:** Возможны конфликты глобальных CSS-стилей (Tailwind) с темой WordPress. Требует изоляции стилей (Shadow DOM).

## 3. Кастомизация внешнего вида (Theming)

Чтобы виджет не выглядел чужеродно на разных сайтах, он поддерживает передачу параметров темы через URL:
- `?theme=dark` / `?theme=light` (Принудительное включение темной/светлой темы)
- `?primaryColor=FF5733` (Замена основного цвета кнопок и акцентов)
- `?hideHeader=true` (Скрытие шапки виджета при встраивании в iframe, чтобы не дублировать логотип клиники)

*Пример:* `https://widget.clinic.ru/?doctorId=12345&hideHeader=true&primaryColor=00A1E4`
