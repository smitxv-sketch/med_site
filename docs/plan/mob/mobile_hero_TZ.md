# ТЗ: Мобильный первый экран — новая концепция
# Клиника «Источник» · Mobile Hero Redesign

**Файл:** `mobile_hero_TZ.md`  
**Статус:** Готово к передаче в Cursor  
**Цель:** Первый экран мобильной версии должен одновременно конвертировать (CTA), навигировать (направления), и дифференцировать (ВРТ как главный продукт). Без перегруза.

---

## 1. Концепция: четыре зоны вместо одного слайдера

Текущий первый экран = слайдер + две карточки (горизонтальный скролл).  
Новый первый экран = **4 компактные зоны**, каждая делает одно дело:

```
┌─────────────────────────────┐  ← хедер (существующий)
├─────────────────────────────┤
│  ЗОНА 1: Слайдер-герой      │  220px, фоновое фото + текст + кнопка
├─────────────────────────────┤
│  ЗОНА 2: Навигация          │  переключатель + чипы направлений
├─────────────────────────────┤
│  ЗОНА 3: Два CTA            │  Записаться | Позвонить
├─────────────────────────────┤
│  ЗОНА 4: Акции (скролл)     │  горизонтальный скролл мини-карточек
├─────────────────────────────┤
│  нижнее меню (существующее) │
└─────────────────────────────┘
```

Всё это должно помещаться на один экран телефона (или почти — акции могут намекать что есть ещё).

---

## 2. Зона 1: Слайдер-герой

**Размеры:** высота 220px, border-radius 14px, margin: 12px 16px 0

**Фон:** полноэкранное фото (object-fit: cover) с градиентным оверлеем:
```css
background: linear-gradient(to top, rgba(0,0,0,0.70) 0%, rgba(0,0,0,0.20) 60%, transparent 100%);
```

**Бейдж направления** (правый верхний угол):
- ВРТ-слайд: `background: rgba(124,58,237,0.85)`, белый текст, «ВРТ · ЦЕНТР ЭКО»
- Поликлиника-слайд: `background: rgba(43,138,62,0.85)`, белый текст
- font-size: 10px, padding: 4px 12px, border-radius: 20px

**Контент (левый нижний угол):**
```
тег (10px, rgba(255,255,255,0.65), letter-spacing: 0.1em)
заголовок (20px, font-weight 500, белый, max 2 строки)
подзаголовок (12px, rgba(255,255,255,0.75), только на широком слайде)
[кнопка CTA] — белый фон, цветной текст (цвет = цвет бейджа направления)
```

**Кнопка CTA на слайдере:**
- background: #ffffff
- color: #7c3aed (ВРТ) или #2b8a3e (поликлиника)
- border-radius: 30px, padding: 8px 20px, font-size: 13px

**Индикаторы-точки:** правый нижний угол, активная — капля (width 18px, border-radius 3px)

**Слайды (порядок жёсткий):**
1. ВРТ — «Лечение бесплодия и программы ЭКО» — бейдж фиолетовый
2. Актуальная акция или новость
3. Оборудование / конференция

---

## 3. Зона 2: Навигация по направлениям

**Контейнер:** background: var(--color-background-secondary), border-radius: 14px, padding: 14px, margin: 12px 16px 0

### Переключатель Поликлиника / ВРТ-клиника
```html
<div class="switcher">
  <button class="active">Поликлиника</button>
  <button>ВРТ-клиника</button>
</div>
```
- Внешний контейнер: белый фон, border-radius 30px, padding 3px
- Активная кнопка: background #2b8a3e, белый текст, border-radius 28px
- Неактивная: прозрачная, цвет текста secondary
- Переключение меняет чипы ниже (JS)

### Чипы направлений

**Поликлиника (по умолчанию):**
- Гинекология, Терапия, Диагностика, УЗИ

**ВРТ-клиника:**
- ЭКО, Ведение беременности, Генетика, Репродуктология

**Стиль чипа:**
- background: var(--color-background-primary)
- border: 0.5px solid var(--color-border-tertiary)
- border-radius: 20px, padding: 6px 14px, font-size: 13px
- Клик → переход на страницу направления

**Кнопка «Все направления · N»:**
- Зелёный текст (#2b8a3e), иконка ti-grid-dots, font-size: 13px
- N = количество направлений активной вкладки

### Анимация переключения
При смене вкладки чипы меняются с fade (opacity 0→1, 150ms ease).

---

## 4. Зона 3: Два CTA

**Layout:** display flex, gap 10px, margin: 12px 16px 0

**Кнопка «Записаться» (основная):**
- background: #2b8a3e, color: #fff
- border-radius: 30px, padding: 13px 0, flex: 1
- Иконка: ti-calendar слева, font-size: 17px
- Действие: открывает виджет онлайн-записи

**Кнопка «Позвонить» (вторичная):**
- background: var(--color-background-secondary)
- border: 0.5px solid var(--color-border-tertiary)
- border-radius: 30px, padding: 13px 0, flex: 1
- Иконка: ti-phone (#2b8a3e) слева, font-size: 17px
- Действие: tel: ссылка

**Важно:** обе кнопки одинаковой высоты, одинакового flex-веса. Не делать одну больше другой.

---

## 5. Зона 4: Акции (горизонтальный скролл)

**Контейнер:** margin: 12px 16px 0

**Лейбл:** «АКЦИИ», font-size 11px, font-weight 500, color secondary, letter-spacing 0.06em

**Скролл-ряд:** display flex, overflow-x auto, gap 10px, padding-bottom 4px  
Скроллбар скрыт: `-webkit-scrollbar { display: none }`

**Мини-карточка акции:**
- min-width: 200px, flex-shrink: 0
- background: var(--color-background-primary)
- border-radius: 12px, padding: 12px
- border: 0.5px solid var(--color-border-tertiary)
- Левая полоска цветом направления: border-left: 3px solid [цвет]

**Контент карточки:**
```
Заголовок акции (13px, font-weight 500, max 2 строки)
Таймер: иконка ti-clock + «Осталось N дней» (11px)
```

**Цвета таймера по срочности:**
- ≤3 дней: #7c3aed (фиолетовый) или #ef4444 (красный)
- 4–7 дней: #f97316 (оранжевый)  
- >7 дней: var(--color-text-secondary)

**Данные:** первые 4 активные акции из коллекции `promotions`, сортировка по `expires_at ASC`

---

## 6. Нижнее меню — не трогать механику

Текущая логика (исчезает при скролле вниз, появляется при движении вверх) — **оставить как есть**. Это стандарт iOS/Android, пользователи знают.

**Единственная правка:** убедиться что весь контент первого экрана имеет `padding-bottom: 70px` чтобы нижнее меню не перекрывало последний элемент при загрузке.

**Кнопка «Запись» в центре нижнего меню:**
- Поднята над панелью на 16px (margin-top: -16px)
- Круглая, background #2b8a3e, border: 3px solid белый
- Это единственный элемент с анимацией — лёгкий pulse (scale 1→1.05→1, 2s infinite) чтобы привлекать внимание

---

## 7. Взаимодействие зон между собой

| Действие пользователя | Результат |
|---|---|
| Переключил на «ВРТ-клиника» | Чипы сменились на ЭКО/Ведение/Генетика/Репродуктология, слайдер переключился на ВРТ-слайд |
| Тапнул на чип «ЭКО» | Переход на страницу направления ЭКО |
| Тапнул «Все направления · 38» | Переход на страницу /services с активной вкладкой Поликлиника |
| Тапнул карточку акции | Переход на страницу акции |
| Тапнул «Записаться» (кнопка зоны 3) | Открывается виджет записи |

---

## 8. Адаптив и крайние случаи

**Если нет активных акций:** зона 4 не рендерится, зона 3 прижимается к зоне 2  
**Если одна акция:** карточка растягивается на 85% ширины экрана  
**Если слайд без фото:** использовать цветной фон (#1a1a2e для ВРТ, #0f3d1f для поликлиники)  
**Если заголовок слайда длиннее 40 символов:** font-size снизить до 18px

---

## 9. Что НЕ делать

- Не добавлять автоскрытие нижнего меню по таймеру
- Не делать кнопку «Записаться» в зоне 3 больше «Позвонить» — равный вес
- Не показывать более 4 чипов одновременно — остальные за кнопкой «Все»
- Не анимировать переключение вкладок переключателя сложнее чем fade
- Не убирать слайдер — он держит визуальный вес верхней зоны

---

## 10. Критерии готовности

- [ ] Все 4 зоны помещаются в один экран iPhone 13 (390×844px) без скролла (или акции намекают на скролл краем карточки)
- [ ] Переключатель Поликлиника/ВРТ меняет чипы с анимацией
- [ ] Цвет кнопки на слайдере соответствует направлению (белая/фиолетовый текст для ВРТ)
- [ ] Таймер в акциях меняет цвет по срочности
- [ ] padding-bottom на странице = высота нижнего меню + 8px
- [ ] Кнопка «Запись» в нижнем меню имеет pulse-анимацию
- [ ] Слайд ВРТ — всегда первый

## Приложение: ASCII-макет финального экрана

```
┌──────────────────────────────┐
│ [лого] ИСТОЧНИК  🔍 📞 ☰    │  48px
├──────────────────────────────┤
│                    [ВРТ·ЭКО] │
│ ЦЕНТР ЭКО                    │
│ Лечение бесплодия            │  220px
│ и программы ЭКО              │  слайдер
│ [Бесплатная консультация]    │
│                     ● ○ ○   │
├──────────────────────────────┤
│ [Поликлиника] [ВРТ-клиника] │
│                              │  ~110px
│ [Гинекология][Терапия][Диаг] │  навигация
│ [УЗИ]  Все направления · 38 →│
├──────────────────────────────┤
│ [📅 Записаться] [📞 Позвонить]│  50px
├──────────────────────────────┤
│ АКЦИИ                        │
│ [Консульт...▌][Скидка 20%...▌│  ~90px
│              → виден край   │
├──────────────────────────────┤
│ 🏠      👨‍⚕️    (📅)   📋   👤  │  56px нижнее меню
└──────────────────────────────┘
  Главная Врачи Запись Услуги ЛК
```

Итого примерно: 48+220+110+50+90+56 = 574px при экране 844px.  
Остаток ~270px — акции чуть выступают вниз, намекая на скролл. ✓
