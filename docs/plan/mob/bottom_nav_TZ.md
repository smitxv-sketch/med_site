# ТЗ: Редизайн нижнего меню (Bottom Navigation)
# Клиника «Источник» · Mobile Bottom Nav

**Файл:** `bottom_nav_TZ.md`  
**Статус:** Готово к передаче в Cursor  
**Контекст:** Нижнее меню — основной инструмент навигации на мобильном. Текущий порядок: Главная / Врачи / Запись / Услуги / ЛК. Задача — переставить приоритеты под реальные сценарии пользователя и усилить визуальный язык.

---

## 1. Что меняем и почему

### Текущий порядок (проблемы)
```
Главная | Врачи | (Запись) | Услуги | ЛК
```
- «Врачи» на втором месте — переоценён как первичный сценарий. Пациент сначала ищет услугу/цену, потом выбирает врача.
- «Услуги» на четвёртом месте — недооценён. Это один из двух главных сценариев захода на сайт.
- Правая зона (Услуги / ЛК) — зона большого пальца правши, самая удобная. Сейчас там второстепенный контент.

### Новый порядок
```
Главная | Услуги и цены | (Запись) | Врачи | ЛК
```

**Логика:**
- Левая зона: Главная — всегда слева, стандарт
- Левее центра: Услуги и цены — первый сценарий («сколько стоит, что есть»)
- Центр: Запись — конверсионная кнопка, самая заметная, без изменений
- Правее центра: Врачи — второй сценарий («хочу к конкретному врачу»)
- Правая зона: ЛК — для лояльных пациентов, им не нужна подсказка

## 2. Спецификация каждого пункта

### Пункт 1 — Главная
- Иконка: `ti-home`
- Подпись: «Главная»
- Действие: navigate('/')
- Активный цвет: #2b8a3e

### Пункт 2 — Услуги и цены
- Иконка: `ti-medical-cross` или `ti-activity`  
  *(не ti-list — слишком абстрактно, нужна медицинская ассоциация)*
- Подпись: «Услуги и цены»
- Действие: navigate('/services')
- Активный цвет: #2b8a3e
- **Примечание:** если подпись «Услуги и цены» не помещается — использовать «Услуги», но НЕ просто «Цены» (теряется смысл каталога)

### Пункт 3 — Запись (центральная кнопка)
- Иконка: `ti-calendar-plus`
- Подпись: «Запись»
- Визуал: круглая кнопка, поднята над панелью
- background: #2b8a3e
- border: 3px solid #ffffff (или цвет фона страницы)
- Размер кнопки: 48px × 48px
- margin-top: -20px (подъём над панелью)
- Анимация: `pulse` — scale(1) → scale(1.06) → scale(1), duration 2s, infinite
  ```css
  @keyframes pulse {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.06); }
  }
  ```
- Действие: открывает виджет онлайн-записи (не navigate, а open widget)
- Подпись цвет: #2b8a3e, font-weight: 600

### Пункт 4 — Врачи
- Иконка: `ti-stethoscope`
- Подпись: «Врачи»
- Действие: navigate('/doctors')
- Активный цвет: #2b8a3e

### Пункт 5 — Личный кабинет
- Иконка: `ti-user-circle`
- Подпись: «ЛК»
- Действие: navigate('/cabinet') или открыть auth modal если не авторизован
- Активный цвет: #2b8a3e
- **Бейдж:** если есть непрочитанные уведомления — красная точка (8px) в правом верхнем углу иконки

## 3. Визуальная спецификация панели

### Размеры
- Высота панели: 56px (без учёта подъёма центральной кнопки)
- Safe area: добавить `padding-bottom: env(safe-area-inset-bottom)` для iPhone с челкой
- Итоговая высота с safe area на iPhone: ~78–82px

### Фон и граница
```css
background: #ffffff;
border-top: 0.5px solid #e8e8e8;
```

### Позиционирование
```css
position: fixed;
bottom: 0;
left: 0;
right: 0;
z-index: 100;
```

### Каждый пункт (кроме центрального)
```css
display: flex;
flex-direction: column;
align-items: center;
gap: 3px;
font-size: 10px;
color: #aaaaaa;  /* неактивный */
cursor: pointer;
flex: 1;
padding: 8px 0 0;
```

**Активный пункт:**
```css
color: #2b8a3e;
```

**Иконка:**
```css
font-size: 22px;
```

### Центральная кнопка — обёртка
```css
flex: 1;
display: flex;
flex-direction: column;
align-items: center;
gap: 3px;
font-size: 10px;
font-weight: 600;
color: #2b8a3e;
padding: 0;
```

## 4. Механика скрытия/показа (не менять)

Текущая логика работает правильно — оставить без изменений:
- При скролле **вниз** (контент уходит вверх): панель скрывается (`transform: translateY(100%)`, transition 200ms ease)
- При скролле **вверх** (пользователь возвращается): панель появляется (`transform: translateY(0)`, transition 200ms ease)
- При загрузке страницы: панель видна
- При открытии виджета записи: панель остаётся видна

**Не добавлять:**
- Автоскрытие по таймеру
- Скрытие при открытии клавиатуры (обрабатывается браузером автоматически)

## 5. Бейдж уведомлений на ЛК

```css
.nav-badge {
  position: absolute;
  top: 6px;
  right: calc(50% - 14px);
  width: 8px;
  height: 8px;
  background: #ef4444;
  border-radius: 50%;
  border: 1.5px solid #ffffff;
}
```

Показывать только если `user.unreadCount > 0`. Если `unreadCount > 9` — показать «9+» (font-size 7px, width auto, padding 0 3px, border-radius 10px).

---

## 6. Отступ контента страницы

Все страницы должны иметь:
```css
padding-bottom: calc(56px + env(safe-area-inset-bottom) + 8px);
```

Это исключает перекрытие контента нижним меню при полном скролле вниз.

---

## 7. Что НЕ менять

- Механику скрытия/показа при скролле — она уже реализована правильно
- Анимацию pulse на кнопке Запись — оставить
- Цветовую схему (#2b8a3e как активный)
- Размер иконок (22px)
