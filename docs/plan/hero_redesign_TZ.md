# ТЗ: Редизайн Hero-блока главной страницы — Клиника «Источник»

**Файл:** `hero_redesign_TZ.md`  
**Статус:** Готово к передаче в Cursor  
**Контекст:** Hero — это первый экран главной страницы. Текущая версия уже использует паттерн «слайдер слева + две плашки справа». Задача — усилить его по дизайну и бизнес-логике, не меняя структуру.

---

## 1. Что сейчас (текущий вариант)

### Слайдер (левая часть)
- Белый фон, фото МРТ-аппарата справа внутри карточки
- Тег «ОБОРУДОВАНИЕ» мелким шрифтом над заголовком
- Заголовок: «Новый аппарат МРТ экспертного класса»
- Кнопка «Подробнее» — зелёная, скруглённая
- Точки-слайды внизу по центру

**Проблема:** фон слайдера белый, фото МРТ занимает правую половину — левая часть визуально пустая. Слайдер не имеет веса, выглядит незавершённым.

### Правые плашки
- Верхняя: «Знакомство с доктором» / Скидка 30% / кнопка «Выбрать врача» / круглое фото врача
- Нижняя: «Подарки для любимых» / Сертификаты / кнопка «Подробности» / фото врача

**Проблема:** нижняя плашка «Подарки для любимых» — нейтральный контент без срочности и без связи с главным differentiator клиники (ВРТ/ЭКО). Теряется ключевое преимущество.

---

## 2. Что нужно сделать

### 2.1 Слайдер — доработка фона и визуального веса

**Задача:** сделать слайдер визуально насыщенным, чтобы он держал левую половину hero.

**Требования:**
- Добавить полноэкранное фоновое изображение на весь слайд (не вставленное справа, а именно фон)
- Текст и кнопка поверх фона с достаточным контрастом
- Для слайда с МРТ: тёмный оверлей (opacity 0.45–0.55) над фото, текст белый
- Для слайдов без фото: использовать цветной фон из палитры бренда (зелёный #2b8a3e или фиолетовый #7c3aed в зависимости от темы слайда)
- Тег над заголовком: сделать контрастнее — белый фон + тёмный текст либо полупрозрачный бейдж
- Точки-индикаторы слайдов: активная точка — вытянутая капля (width 24px, border-radius 4px), остальные — круги

**Размеры слайдера:** минимальная высота 380px, border-radius 16px

---

### 2.2 Верхняя правая плашка — оставить, усилить

**Текущий контент:** «Знакомство с доктором», скидка 30%, кнопка «Выбрать врача», фото врача

**Что оставить:** структуру и контент  
**Что усилить:**
- Фото врача сделать чуть крупнее (52×52px вместо текущего ~40px), обрезка по кругу
- Добавить под именем врача специализацию (если данные доступны из CMS)
- Кнопка «Выбрать врача» — зелёная заливка (#2b8a3e), белый текст, border-radius 30px
- Фон плашки: белый с тонкой рамкой (0.5px, #e0e0e0)

---

### 2.3 Нижняя правая плашка — ЗАМЕНИТЬ контент

**Текущий контент:** «Подарки для любимых» / Сертификаты  
**Заменить на:** акцент на ВРТ/ЭКО — главный differentiator клиники

**Новый контент плашки:**

```
Метка (сверху):    ВРТ · ЦЕНТР ЭКО
Заголовок:         Бесплатная консультация репродуктолога
Подзаголовок:      Осталось 1 день
Кнопка:            Записаться
Ссылка под кнопкой: → Все программы ЭКО
```

**Визуальное оформление:**
- Фон плашки: светло-фиолетовый (#f0ebfa)
- Рамка: #c9b8f0 (0.5px)
- Метка «ВРТ · ЦЕНТР ЭКО»: цвет #7c3aed, font-size 10px, font-weight 500, letter-spacing 0.08em
- Заголовок: цвет var(--color-text-primary), font-size 16px, font-weight 500
- Подзаголовок «Осталось N дней»: цвет #7c3aed, font-size 12px — данные тянуть из того же источника, что и блок Акций (поле срока акции)
- Кнопка «Записаться»: фон #7c3aed, белый текст, border-radius 30px, font-size 13px
- Ссылка «→ Все программы ЭКО»: цвет #7c3aed, font-size 12px, без подчёркивания, hover — underline
- Круглое фото: аватар репродуктолога (если есть в CMS) либо иконка-заглушка

**Бизнес-логика:** эта плашка должна быть динамической. Показывать текущую активную акцию с тегом «ВРТ» из раздела Акций. Если активной ВРТ-акции нет — fallback на статичный контент «Центр ЭКО — узнать о программах».

## 3. Динамика и CMS-логика

| Элемент | Источник данных |
|---|---|
| Слайды (заголовок, тег, фото, ссылка) | Коллекция `hero_slides` в Strapi |
| Верхняя плашка (врач, скидка) | Коллекция `hero_cards` или хардкод с ручным редактированием |
| Нижняя плашка (ВРТ-акция) | Автоматически: фильтр по тегу `ВРТ` в коллекции `promotions`, сортировка по `expires_at ASC`, первый результат |
| Счётчик дней в плашке | `Math.ceil((expires_at - now) / 86400000)` |
| Fallback если нет ВРТ-акции | Статичный блок: «Центр ЭКО — узнать о программах» |

---

## 4. Адаптив (мобильная версия)

- Breakpoint: 768px
- На мобильном: правые плашки уходят вниз под слайдер
- Layout: `grid-template-columns: 1fr`
- Слайдер: высота 260px
- Плашки: горизонтальный скролл (overflow-x: auto, display: flex, gap: 12px, каждая плашка min-width: 260px)

---

## 5. Что НЕ менять

- Хедер (навигация, телефон, кнопка «Записаться»)
- Блок Акций ниже hero — он уже сильный
- Блок Направлений с ВРТ — уже хорошо
- Цветовую систему бренда (зелёный #2b8a3e как основной)

---

## 6. Критерии готовности

- [ ] Слайдер имеет фоновое изображение на всю карточку, текст читается на любом фоне
- [ ] Нижняя правая плашка показывает актуальную ВРТ-акцию с таймером дней
- [ ] Если ВРТ-акций нет — корректно показывается fallback
- [ ] На мобильном плашки скроллятся горизонтально, не ломают layout
- [ ] Высоты двух правых плашек в сумме равны высоте слайдера (без разрыва)
- [ ] Фиолетовый цвет ВРТ-плашки визуально рифмуется с фиолетовым блоком в секции Направлений ниже
